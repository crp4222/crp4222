## 🧑‍💻 About me

I'm a **Data / Software Engineer** working on **privacy-first tooling for LLMs** — keeping personal data out of AI calls.

- 🛡️ Building **PrivAiTe**, a self-hosted proxy that redacts PII before it reaches model providers.
- 🌱 Deep into **NER, ONNX runtime, and data pipelines** — everything around trustworthy AI.
- 💬 Ask me about **LLM proxies, PII detection, Python, and data engineering**.

## 🚀 Featured Project — PrivAiTe

<table>
  <tr>
    <td width="120" align="center" valign="middle">
      <br/>
      🛡️<br/>
      <b>PrivAiTe</b>
    </td>
    <td valign="middle">
      A drop-in, self-hosted <b>LLM proxy that reversibly redacts PII</b> — names, emails, cards, IBANs, secrets — before it reaches OpenAI, ChatGPT or Ollama.
      It works <b>inside tool-call arguments and multimodal content</b>, restores the real values in the reply, is OpenAI-compatible and ships with <b>zero telemetry</b>.
